# Фаза Merge (Злиття)

Ця фаза буде реалізована для злиття часткових суфіксних дерев у єдине узагальнене суфіксне дерево.

## План реалізації

1. Групування S-префіксів за ключами
2. Злиття часткових дерев на основі спільних префіксів
3. Побудова фінального узагальненого суфіксного дерева
4. Оптимізація структури дерева
